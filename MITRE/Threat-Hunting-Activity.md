# Threat Hunting Activity Metrics

## Purpose

Measure proactive threat hunting activities performed to identify suspicious behavior and improve detection capabilities.

## Metrics

| Metric | Description |
|----------|-------------|
| Hunting Queries Executed | Total number of threat hunting queries performed. |
| Suspicious Findings | Number of suspicious activities identified during hunting. |
| Validated Threats | Findings confirmed as security threats after investigation. |
| MITRE ATT&CK Techniques Investigated | ATT&CK techniques covered during threat hunting activities. |

## Data Source

- Microsoft Sentinel Hunting Queries
- Log Analytics Workspace
- MITRE ATT&CK Mapping

## Objective

These metrics help evaluate the effectiveness of proactive threat hunting, improve detection coverage, and identify opportunities to strengthen the security monitoring environment.
