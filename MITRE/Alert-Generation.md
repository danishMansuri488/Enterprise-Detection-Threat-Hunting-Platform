# Alert Generation Metrics

## Purpose

Track security alerts generated by Microsoft Sentinel to measure detection activity and identify trends.

## Metrics

| Metric | Description |
|----------|-------------|
| Total Alerts | Total number of alerts generated during the monitoring period. |
| Alert Severity | Distribution of High, Medium, Low, and Informational alerts. |
| Alerts by Rule | Number of alerts generated by each analytics rule. |
| Top Triggered Rules | Analytics rules that generated the highest number of alerts. |

## Data Source

- Microsoft Sentinel
- Analytics Rules

## Objective

These metrics help evaluate detection effectiveness, identify noisy rules, and support continuous tuning of analytics rules.
