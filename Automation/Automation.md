# Automation

## Overview

This project includes automation concepts to improve Security Operations Center (SOC) efficiency within Microsoft Sentinel.

## Automation Objectives

- Reduce repetitive analyst tasks.
- Improve incident response efficiency.
- Standardize security operations.

## Planned Automation

- Incident Tagging
- Severity Assignment
- Alert Enrichment
- Known-Good Baseline Identification
- Email Notifications
- Microsoft Sentinel Playbooks (Logic Apps)

## Benefits

- Faster incident triage
- Consistent response workflow
- Reduced manual effort
- Improved operational efficiency
