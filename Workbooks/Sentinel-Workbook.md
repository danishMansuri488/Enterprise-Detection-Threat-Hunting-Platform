# Microsoft Sentinel Workbook

## Overview

The Microsoft Sentinel Workbook provides a centralized dashboard for monitoring authentication activity and security events collected from Windows endpoints and Active Directory.

## Dashboard Components

- Total Successful Logons
- Total Failed Logons
- Account Lockouts
- Failed Logon Trend
- Top Failed Login Accounts
- Top Source Computers
- Login Activity Timeline

## Purpose

The workbook enables SOC analysts to quickly identify abnormal authentication behavior, investigate suspicious activity, and monitor the overall security posture of the environment.

## Data Source

- Microsoft Sentinel
- Log Analytics Workspace
- Windows Security Events
- Active Directory
- Sysmon

## Benefits

- Centralized visibility
- Faster incident investigation
- Improved threat detection
- Real-time monitoring
