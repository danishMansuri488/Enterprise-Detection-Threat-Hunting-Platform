# Metrics

## Overview

The Metrics documentation provides a structured approach for measuring the effectiveness of detection engineering, threat hunting, and incident response activities within the Enterprise Detection & Threat Hunting Platform.

These metrics help evaluate detection coverage, monitor operational performance, identify improvement opportunities, and support continuous enhancement of Microsoft Sentinel security monitoring.

---

## Objectives

- Measure detection effectiveness.
- Evaluate incident response performance.
- Monitor threat hunting activities.
- Assess detection coverage.
- Support continuous improvement.

---

## Documentation

| Document | Description |
|----------|-------------|
| Detection-Coverage.md | Measures coverage of implemented detections and monitored attack techniques. |
| Detection-Performance.md | Evaluates detection accuracy and operational performance. |
| Incident-Metrics.md | Tracks incident handling and response effectiveness. |
| Threat-Hunting-Metrics.md | Measures proactive threat hunting activities and outcomes. |

---

## Data Sources

- Microsoft Sentinel
- Azure Log Analytics Workspace
- Windows Security Events
- Sysmon
- Active Directory

---

## Purpose

These metrics provide measurable insight into the overall effectiveness of the security monitoring environment and support ongoing improvement of detection engineering and SOC operations.
